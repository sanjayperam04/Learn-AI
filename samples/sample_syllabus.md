# React Development Course - Syllabus

## Course Overview
This 8-week course covers React fundamentals and advanced concepts for building modern web applications.

## Week 1-2: React Fundamentals
- Introduction to React and JSX
- Components and Props
- State and Lifecycle
- Event Handling
- Conditional Rendering
- Lists and Keys

## Week 3-4: Hooks and Modern React
- Introduction to Hooks
- useState and useEffect
- useContext and useReducer
- Custom Hooks
- Rules of Hooks

## Week 5-6: Advanced Patterns
- Context API
- Higher-Order Components
- Render Props
- Component Composition
- Error Boundaries

## Week 7: Routing and Navigation
- React Router basics
- Dynamic routing
- Nested routes
- Navigation guards
- URL parameters

## Week 8: Performance and Testing
- Performance optimization techniques
- React.memo and useMemo
- Code splitting
- Testing with Jest
- React Testing Library

## Learning Objectives
By the end of this course, students will be able to:
1. Build complex React applications
2. Understand component lifecycle and hooks
3. Implement routing and navigation
4. Optimize application performance
5. Write tests for React components
6. Follow React best practices

## Assessment
- Weekly coding assignments (40%)
- Mid-term project (30%)
- Final project (30%)
